Tujuannya bukan meniru Canva mentah-mentah, tapi mengambil pola layout: app shell, navigation rail, sidebar, hero search, carousel, grid, list/table, modal, popover, drawer, dan empty state.

Resep layout siap pakai

Ambil pola besar Canva, lalu adaptasi untuk project berbeda.

Dashboard / Semua Proyek

  1. App Shell: rail kiri + area utama.
  2. Page Header: title + search besar + filter chips.
  3. Recent Strip: carousel desain terbaru.
  4. Data View: table/list untuk file lengkap.
  5. Actions: sort, grid/list toggle, create button.

Create Flow / Buat Desain

  1. Trigger dari primary action.
  2. Backdrop menggelapkan layar lama.
  3. Modal besar berisi search command.
  4. Left Category Nav untuk kategori format.
  5. Template tiles dan quick actions sebagai isi modal.

Template Gallery

  1. Hero search untuk eksplorasi.
  2. Category chips di bawah search.
  3. Browse category grid.
  4. Section carousel untuk rekomendasi.
  5. CTA “lihat semua” per section.

Assistant / AI Panel

  1. Secondary sidebar untuk chat history.
  2. Main empty state dengan command input.
  3. Mode chips: Desain, Gambar, Doc, Kode.
  4. Right assistant panel jika butuh bantuan kontekstual.
  5. Disclaimer kecil untuk akurasi AI.
Prinsip cepat: Card menampilkan satu objek. Section mengelompokkan banyak card. Container membungkus section. Page menata beberapa container. App Shell membungkus seluruh aplikasi.

Hirarki container yang aman dipakai di semua project

Gunakan ini saat bikin wireframe, PRD frontend, atau prompt HTML/React.

AppShell ├─ NavigationRail / PrimarySidebar │ ├─ BrandArea │ ├─ PrimaryNavGroup │ │ └─ NavItem │ └─ UserDock / UtilityDock ├─ SecondarySidebar optional │ ├─ SidebarHeader │ ├─ SidebarSection │ │ └─ MenuItem / MiniListItem │ └─ SidebarFooter ├─ MainRegion │ ├─ PageHeader / HeroHeader │ │ ├─ PageTitle │ │ ├─ GlobalSearch / CommandInput │ │ └─ FilterBar / QuickActionBar │ ├─ ContentSection │ │ ├─ SectionHeader │ │ └─ CardGrid / Carousel / DataTable / ListView │ └─ FloatingActionButton / HelpButton ├─ RightPanel optional │ ├─ PanelHeader │ ├─ PanelBody │ └─ PanelInput / PanelFooter └─ OverlayLayer optional ├─ Backdrop ├─ ModalDialog / Drawer / Popover / ContextMenu └─ CloseButton

Catatan: istilah di atas adalah nama pola UI umum. Untuk design system, lebih baik konsisten memakai nama Inggris seperti NavigationRail, ContentSection, TemplateCard, dan CommandInput, lalu beri label Indonesia untuk dokumentasi user-facing.
